# 자동차 경주 - 기능목록

## 🕹️️ 기능 목록
___

- [ ] 경주할 자동차의 이름을 입력 받습니다.
  - [ ] 자동차의 이름은 "," 기준으로 구분합니다.
  - [ ] 이름은 5자 이하로 제한합니다.
  - [ ] 입력 조건에 맞지 않을 경우 IAE를 발생시키고 애플리케이션을 종료합니다.


- [ ] 시도할 회수를 입력 받습니다.


- [ ] 입력받은 숫자만큼 게임을 진행합니다.
  - [ ] 각 자동차는 조건에 따라 전진합니다.
  - [ ] 조건: 0에서 9 사이의 무작위 값을 구하고 4 이상일 경우 전진합니다.


- [ ] 각 차수별 실행 결과를 보여줍니다.


- [ ] 게임이 마무리 되면 우승 여부를 파악합니다.


- [ ] 우승 여부는 전진한 숫자가 가장 큰 자동차입니다. (여러명이 우승할 수 있습니다.)
